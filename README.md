# 마파이빙

링크, 이미지, 메모를 한 곳에 저장하는 개인용 아카이빙 웹앱입니다.

## 현재 범위

| 구분 | 내용 |
| --- | --- |
| 할 일 | 체크, 수정, 순서 변경, 삭제, 여러 줄 자동 추가 |
| 저장 | 링크, 이미지, 메모 저장 |
| 정리 | 폴더 생성, 삭제, 폴더별 보기 |
| 탐색 | 제목, 메모, 태그, 출처 검색 |
| 지표 | 저장 수, 재조회 수, 연속 저장일 |
| 동기화 | Cloudflare KV 기반 자동 저장 |
| 안내 | 온보딩, 공지사항 페이지 |

## 실행

```bash
npm install
npm run dev
```

동기화 API를 로컬에서 별도로 테스트하려면 Cloudflare 로그인 후 아래 명령을 사용합니다.

```bash
npm run dev:api
```

## 배포

프론트는 GitHub Pages 배포가 기본입니다. `main` 브랜치에 올리면 GitHub Actions가 자동으로 빌드하고 배포합니다.

동기화 백엔드는 Cloudflare Worker와 Workers KV를 사용합니다.

데이터는 브라우저 저장소에 먼저 저장되고, 연결된 Cloudflare KV에도 자동 저장됩니다.
