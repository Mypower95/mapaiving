# 마파이빙

링크, 이미지, 메모를 한 곳에 저장하는 개인용 아카이빙 웹앱입니다.

## 현재 범위

| 구분 | 내용 |
| --- | --- |
| 할 일 | 체크, 수정, 순서 변경, 삭제, 여러 줄 자동 추가 |
| 저장 | 링크, 이미지, 메모 저장 |
| 정리 | 폴더 생성, 삭제, 폴더별 보기 |
| 탐색 | 제목, 메모, 태그, 출처 검색 |
| 지표 | 저장 수, 재조회 수, 연속 저장일 |
| 안내 | 온보딩, 공지사항 페이지 |

## 실행

```bash
npm install
npm run dev
```

## 배포

GitHub Pages 배포가 기본입니다. `main` 브랜치에 올리면 GitHub Actions가 자동으로 빌드하고 배포합니다.

현재 데이터는 브라우저 저장소에 보관됩니다. 기기 간 동기화, 로그인, 장기 파일 보관은 별도 서버와 스토리지가 필요합니다.
